# vision - 视觉分析

## 目录说明

本目录包含高层视觉分析代码，基于 C++ L1 层的输出进行进一步分析。

## 文件说明

### enhanced_gui_matcher.py
增强版 GUI 元素匹配

### game_state_detector.py
游戏状态机识别

### gui_matcher.py
基础 GUI 元素匹配

### squad_analyzer.py
编队分析

### squad_recognizer.py
干员识别（接收 C++ L1 输出）

### text_locator.py
文本定位

## 相关目录

- [inference/src/data/](../data/): 数据层
- [inference/src/map/](../map/): 地图分析
