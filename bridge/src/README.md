# bridge/src - 桥接层实现

## 目录说明

本目录包含 C++ ↔ Python 桥接层的具体实现代码。

## 文件说明

### grpc_client.cpp
C++ gRPC 客户端实现:
- 连接管理
- 请求/响应序列化
- 流处理

### shm_segment.cpp
共享内存段实现:
- Boost.Interprocess 封装
- 环形缓冲区管理
- 进程间同步

### pybind_module.cpp
pybind11 模块定义:
- Python 可调用的 C++ API
- 类型转换
- 异常处理
