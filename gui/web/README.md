# web - Web 远程控制台（可选）

## 目录说明

本目录包含可选的 Web 远程控制台实现，基于 WebAssembly 或 Electron。

## 用途

- 配置编辑
- 战报查看
- 远程监控（通过 WebRTC）

## 技术栈（预留）

- **WebAssembly**: C++ 核心编译到 WASM
- **React/Vue**: 前端框架
- **WebRTC**: 实时通信

## 状态

⚠️ 此模块为预留扩展，当前版本未实现。

## 相关目录

- [gui/qt/](../qt/): Qt6 实现（主推）
- [gui/wpf/](../wpf/): WPF 实现（Windows）
