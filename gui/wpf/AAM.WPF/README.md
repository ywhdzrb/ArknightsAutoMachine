# AAM.WPF - WPF 主项目

## 目录说明

本目录包含 WPF GUI 的 C# 主项目。

## 目录结构

```
AAM.WPF/
├── Views/              # XAML 视图
│   ├── MainWindow.xaml
│   └── MapOverlay.xaml
├── ViewModels/         # MVVM 视图模型
└── App.xaml            # 应用入口
```

## 技术栈

- .NET 8
- WPF
- MVVM 模式
